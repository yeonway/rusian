# Rusian Learner Web

Browser version of the Android learning app. It reads the same `content-pack.json`
and stores progress in `localStorage`.

## Run

```powershell
npm start
```

Open:

```text
http://localhost:4173
```

## Included

- Alphabet board with browser text-to-speech
- Word flashcards with local review progress
- Meaning quiz
- Dataset search chat
- Theme, daily goal, and progress reset settings
